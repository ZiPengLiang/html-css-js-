CSS3 渐变（Gradients）
  • 线性渐变（Linear Gradients）-- 向下/向上/向左/向右/对角方向
  • 径向渐变（Radial Gradients）-- 由它们的中心定义

线性渐变 - 从上到下（默认情况下）

默认写法：background: linear-gradient(rgb(250, 26, 10), rgb(7, 248, 7))

线性渐变 - 从左到右

标准写法：background: linear-gradient(to right, red, blue)

线性渐变 - 对角（从左上到右下）

标准写法：background: linear-gradient(to bottom right, yellow, green)

线性渐变 - 角度（0deg）

标准写法： background: linear-gradient(0deg, white, black)

线性渐变 - 角度（90deg）

标准写法： background: linear-gradient(90deg, white, black)

线性渐变 - 角度（180deg）

标准写法： background: linear-gradient(180deg, white, black)

线性渐变 - 角度（-90deg）

标准写法： background: linear-gradient(-90deg, white, black)

线性渐变 - 颜色百分比（等分）

标准写法：background: linear-gradient(red, yellow, blue)

线性渐变 - 颜色百分比（等分--彩虹背景）

标准写法： background: linear-gradient(to right, red, orange, yellow, green, blue, indigo, violet)

线性渐变 - 颜色百分比（不等分）

标准写法：background: linear-gradient(red 10%, yellow 20%, blue 70%)

线性渐变 --透明度

标准写法： background: linear-gradient(to right, rgba(255, 0, 0, 0), rgba(255, 0, 0, 1))

线性渐变 --重复线性渐变

标准写法： background: repeating-linear-gradient(red, yellow 10%, green 20%)

径向渐变

标准写法： background: radial-gradient(red, yellow, green)

径向渐变 -- 颜色不均

标准写法： background: radial-gradient(red 10%, yellow 30%, green 60%)

径向渐变 -- 设置形状

标准写法：background: radial-gradient(circle, red, yellow, green)

径向渐变 -- 关键字（没有关键字）

标准写法： background: radial-gradient(green, yellow, red)

径向渐变 -- 关键字（closest-side）

标准写法： background: radial-gradient(closest-side, green, yellow, red)

径向渐变 -- 关键字（farthest-side）

标准写法： background: radial-gradient(farthest-side, green, yellow, red)

径向渐变 -- 关键字（closest-corner）

标准写法： background: radial-gradient(closest-corner, green, yellow, red)

径向渐变 -- 关键字（farthest-corner）

标准写法： background: radial-gradient(farthest-corner, green, yellow, red)

径向渐变 -- 不同大小(正常)

标准写法：background: radial-gradient( red, yellow, orange)

径向渐变 -- 不同大小(60% 40%)

标准写法：background: radial-gradient(60% 40%, red, yellow, orange)

径向渐变 -- 重复的径向渐变

标准写法： repeating-radial-gradient(red 10%, yellow 15%, green 15%)
